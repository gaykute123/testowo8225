    Ví dụ:

    Output:

    Hướng dẫn cách làm:

    Sử dụng System.out.println để in dòng văn bản và tự động xuống dòng sau mỗi lần in. Sử dụng System.out.print để in mà không xuống dòng, các dòng kế tiếp sẽ in tiếp vào cùng một dòng hiện tại.

    Các bước thực hiện:

    1. Sử dụng System.out.println để in dòng đầu tiên, sau đó tự động xuống dòng.
    2. Sử dụng System.out.print để in dòng thứ hai và giữ trên cùng một dòng.

    Hướng dẫn viết code:

    1. ▶ Sử dụng System.out.println để in và xuống dòng 
      
public class PrintExample {
    public static void main(String[] args) {
        System.out.println("Hello World!");
        System.out.println("Đây là một dòng mới!");
    }
}
    2. ▶ Sử dụng System.out.print để in trên cùng một dòng 
      
public class PrintExample {
    public static void main(String[] args) {
        System.out.print("Hello World! ");
        System.out.print("Đây là một dòng tiếp theo!");
    }
}

    Các bước thực hiện:

    1. Viết một chương trình Java đơn giản với lỗi cú pháp (thiếu dấu chấm phẩy ;).
    2. Biên dịch và kiểm tra thông báo lỗi từ trình biên dịch.

    1. ▶ Viết mã có lỗi cú pháp

      Chương trình dưới đây bị thiếu dấu ; ở cuối câu lệnh System.out.println.

      
public class SyntaxErrorExample {
    public static void main(String[] args) {
        System.out.println("Hello, world!") // Lỗi: Thiếu dấu chấm phẩy
    }
}
    2. ▶ Sửa lỗi cú pháp

      Thêm dấu chấm phẩy ; vào cuối câu lệnh để sửa lỗi.

      
public class SyntaxErrorExample {
    public static void main(String[] args) {
        System.out.println("Hello, world!"); // Đã sửa lỗi
    }
}

    [2 điểm] Bài 1.7: Tạo một chương trình với nhiều phương thức để thực hiện các chức năng khác nhau

    1. Nhập một số từ người dùng
    2. Tìm tất cả các ước số của số đó
    3. In ra các ước số

    Các bước thực hiện:

    1. Phương thức main: Tạo phương thức main để gọi các phương thức khác theo trình tự.
    2. Phương thức nhập số: Tạo phương thức nhapSo để yêu cầu người dùng nhập vào một số nguyên dương.
    3. Phương thức tìm ước số: Tạo phương thức timUoc để tính và trả về danh sách các ước số của số đó.
    4. Phương thức in ước số: Tạo phương thức inUoc để in ra danh sách các ước số đã tìm được.

    Pseudocode:

    Phương thức chính (main):
    - Gọi nhapSo để nhập một số nguyên dương n từ người dùng
    - Gọi timUoc với tham số n để tìm danh sách các ước số
    - Gọi inUoc với danh sách ước số để in kết quả

Phương thức nhapSo:
    - Hiển thị yêu cầu người dùng nhập số nguyên dương
    - Đọc và trả về số đã nhập

Phương thức timUoc(n):
    - Khởi tạo danh sách rỗng `dsUoc`
    - Với mỗi i từ 1 đến n:
        - Nếu n chia hết cho i, thêm i vào `dsUoc`
    - Trả về `dsUoc`

Phương thức inUoc(dsUoc):
    - In "Các ước số của n là:" và danh sách `dsUoc`

    Hướng dẫn viết code:

    1. ▶ Viết phương thức nhapSo để yêu cầu người dùng nhập số

public class MultipleMethods {
    public static int nhapSo() {
        Scanner scanner = new Scanner(System.in);

        int n = scanner.nextInt();
        scanner.close();
        return n;
    }
}
    2. ▶ Viết phương thức timUoc để tìm các ước số của n

public static String timUoc(int n) {
    StringBuilder dsUoc = new StringBuilder();
    for (int i = 1; i <= n; i++) {
        if (n % i == 0) {
            dsUoc.append(i).append(", ");
        }
    }
    return dsUoc.toString();
}

public static void main(String[] args) {
    int n = nhapSo();
    String uoc = timUoc(n);

    3. Kiểm tra điều kiện chia hết và tính chẵn lẻ: Trong mỗi bước lặp, chúng ta kiểm tra xem i có phải là ước của n (tức n % i == 0) và có phải là số chẵn (i % 2 == 0) hay không.

import java.util.Scanner;

public class Rectangle {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("Nhập chiều dài m: ");
        int m = scanner.nextInt();
        System.out.print("Nhập chiều rộng n: ");
        int n = scanner.nextInt();
        
        System.out.println("Hình chữ nhật đặc:");
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                System.out.print("*");
            }
            System.out.println();
        }

        System.out.println("Hình chữ nhật rỗng:");
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (i == 1 || i == m || j == 1 || j == n) {
                    System.out.print("*");
                } else {
                    System.out.print(" ");
                }
            }
            System.out.println();
        }
        
        scanner.close();
    }
}
